# Local LLM Config

This repository provides configurations for running local Large Language Models
(LLMs) with the [Pi coding agent](https://pi.dev) and [llama.cpp](https://github.com/ggml-org/llama.cpp).

The [AMD Strix Halo Llama.cpp Toolboxes](https://github.com/kyuz0/amd-strix-halo-toolboxes) were used
or running llama.cpp, and comes highly recommended.

## Repository Structure

- `agent/`: Pi coding agent configuration.
- `models/`: llama.cpp model configuration files.
